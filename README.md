Fluxy is a self-hosted gaming platform for running and organizing your own HTML game library. It has a React frontend, a Node/Express backend, Socket.IO chat, and admin tooling.

What The Site Is About
Launch local HTML games from one place
Manage game metadata in JSON
Track play counts and usage
Provide chat rooms and admin tools for your community
By default, games are served from: Fluxy/client/UGS Files

Stack
Frontend: React (Create React App)
Backend: Node.js + Express
Realtime: Socket.IO
Data: JSON files in server/db
